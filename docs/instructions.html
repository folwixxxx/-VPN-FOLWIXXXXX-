Подключение на разных устройствах
⚠️ Важно! Ссылка на подписку выдается в боте после оплаты. Получить её можно в разделе «Профиль» или после покупки.
🤖
Android (v2rayNG)
Шаг 1
Скачайте v2rayNG из Google Play или GitHub
Шаг 2
Откройте приложение → нажмите на плюс ➕ в правом верхнем углу
Шаг 3
Выберите «Импорт подписки из буфера» или «Добавить подписку»
Шаг 4
Вставьте ссылку, которую дал бот → нажмите «ОК»
Шаг 5
Нажмите на иконку ▶️ (треугольник) для подключения
📥 Скачать: GitHub | Google Play
🍎
iOS (iPhone / iPad)
Вариант A — NekoBox (бесплатно)
Скачайте NekoBox из App Store → Добавьте подписку → Вставьте ссылку
Вариант B — Shadowrocket (платный)
Скачайте Shadowrocket → Нажмите «+» → «Subscribe» → Вставьте ссылку
Вариант C — FoXray (бесплатно)
Скачайте FoXray → Добавьте подписку через + → укажите ссылку
🪟
Windows (v2rayN / NekoRay)
Шаг 1
Скачайте v2rayN или NekoRay с GitHub
Шаг 2
Распакуйте архив и запустите программу
Шаг 3
Нажмите «Подписки» → «Добавить подписку» → Вставьте ссылку
Шаг 4
ПКМ на сервере → «Выбрать активный сервер» → Включите прокси в трее
📥 Скачать: v2rayN | NekoRay
🍏
macOS (NekoRay)
Шаг 1
Скачайте NekoRay для macOS с GitHub
Шаг 2
Запустите, нажмите «Подписки» → «Добавить подписку»
Шаг 3
Вставьте ссылку от бота, сохраните
Шаг 4
Включите «Tun» или «Системный прокси»
💡 Советы:
• Если не работает — попробуйте удалить подписку и добавить заново
• Обновляйте подписку после продления (нажмите на подписку → «Обновить»)
• Для просмотра даты истечения используйте команду /profile
